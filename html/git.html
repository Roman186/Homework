Основные команды Git

  • git init - Создание нового репозитория в пустой директории.
  • git clone "URL" - Клонирование удаленного репозитория из удаленного хранилища на локальную машину.
  • git push - Выгрузка сделанных изменений в удаленный репозиторий.
  • git pull - Получение изменений из удаленного репозитория.
  • git commit - Сохранение изменений в git.
  • git status - Просмотр списка изменненых файлов.
  • git add - Добавление файлов в репозиторий.
  • git reset - Отменяет изменения в файлах в рабочем каталоге.
  • git branch - Создание новой ветки
  • git checkout - Переход на ветку.
  • git log - Просмотр истории коммитов.
